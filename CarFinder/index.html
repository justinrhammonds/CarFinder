Special Features


Built using ASP.NET and SqlServer

The CarFinder Web API is built using SqlServer (T-SQL) and the ASP.NET (C#) framework. The result is a powerful web service that provides valuable information for a large number of vehicles. This information could be useful in automotive performance and safety testing, car manufacturing and sales, and in general research about vehicle specifications.

Designed with Angular JS + trNgGrid

The CarFinder Web API application is designed using AngularJS on the front end utilizing HTML, CSS, and JavaScript. The grid displaying the table data is constructed using the trNgGrid plugin.

Fully Responsive Web App

The CarFinder Web API application is designed to look great and function well in any screen size, whether on your mobile phone, a tablet or a desktop computer.
